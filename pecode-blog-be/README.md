## Requirements
- Node 18+

## 1. Backend

1. cd pecode-blog-be
2. cp .env.example .env
3. npm i
4. npm run db:create
5. npm run migration:run
6. npm run test (optional)
7. npm run start

## 2. Frontend

1. cd pecode-blog-ui
2. cp .env.example .env
3. npm i
4. npm run build && npm start

### Swagger - http://localhost:4000/api#/
